IV. MEM TAGGING FORMAT
────────────────────────────────────────────────────────────
MEMs may include an optional CONCEPTS section:

```
────────────────────────────────────────────────────────────
CONCEPTS (SEMANTIC INDEX)
────────────────────────────────────────────────────────────
Primary:
• legitimacy_through_suffering — Peter's reforms create suffering later harvested
• coercive_enforcement — Modernization requires state violence

Secondary:
• corridor_control — Baltic access as strategic objective
• elite_overproduction — Western-trained elites without absorption pathway
```

REQUIREMENTS:
• 2-4 concepts per MEM (avoid over-tagging)
• Each tag includes one-line explanation
• Primary = central to MEM's analysis
• Secondary = relevant but not central
