# Russia MEMs to Refine for RLL–0034 (Legitimacy Sequencing) Alignment

**Date:** 16 February 2026  
**Trigger:** RLL–RUSSIA–0034 (Legitimacy Sequencing Law — Rurikid → Horde → Muscovite) bound; STATE Section VII pattern "Legitimacy Sequencing (Rurikid → Horde → Muscovite)" added.
**Purpose:** Select Russia MEMs that should be refined to reference or align with RLL–0034 and the four-step legitimacy sequence (order-restoration → sacral/legal ratification → licensed rule under overlordship → internalisation when overlordship fragments).

**Implementation (2026-02-16):** Refinements applied to all 8 selected MEMs per format below. Additive only; no analytical content changed.

---

## Selection criteria

- **Direct evidence for RLL–0034** — MEM is a derivation or evidence source for the bound law.
- **Formation / sovereignty / settlement** — MEM encodes legitimacy claims, succession, or authority conversion (settlement, post-war recognition, election-after-chaos).
- **Gap** — MEM does not yet reference RLL–0034, or its continuity insights do not yet frame content in the four-step sequence.
- **STATE relevance** — MEM is cited in STATE Section VII (Decision-Relevant History) or in MEM–RELEVANCE under dimensions where legitimacy sequencing applies (e.g. Settlement Precedent, Leadership Liability).
