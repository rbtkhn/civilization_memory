I. INDEX PURPOSE & SCOPE
────────────────────────────────────────────────────────────
This index enumerates all **Rome-related Memory (MEM) files** within the
Civilizational Memory Codex. These files collectively preserve Rome not
as a single regime, but as a **multi-form civilizational continuum**:
republican, imperial, ecclesiastical, literary, and metaphysical.

Entries are listed for navigational clarity only.
