I. INDEX PURPOSE & SCOPE
────────────────────────────────────────────────────────────
This index enumerates all Rome-related Memory (MEM) files within the
Civilizational Memory Codex. These files preserve Rome not as a single
state or regime, but as a persistent civilizational continuum spanning
republican, imperial, Christian, imperial-successor, and symbolic forms.

Rome is treated here as a transmissible system, not a terminal polity.
Interpretation is contained within individual MEM files; this index
functions solely as authoritative navigation.
