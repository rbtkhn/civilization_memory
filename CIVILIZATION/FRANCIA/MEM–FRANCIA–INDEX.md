This file is the authoritative register of all MEM–FRANCIA files.

INDEX GOVERNANCE RULE (CROSS-CIV COMPARABILITY):
1) Canonical Sections (mandatory, same order for every civilization index):
   A. GEO
   B. DYNASTY
   C. WAR
   D. LAW
   E. ECON
   F. RELIGION
   G. PERSON (ONLY if used by that civilization’s naming convention)
   H. OTHER / MISC (sparingly; requires promotion plan)

2) Sorting Rule:
   • Within each section: sort by earliest primary date/event first.
   • If dates are equal/unclear: alphabetical.

3) Entry Schema (mandatory):
   • Filename
   • Dates
   • One-line function (what the file preserves)

4) Integrity Rule:
   • Any new MEM–FRANCIA file must be added here the same day it is created.
   • Any rename must be reflected here immediately (index is source of truth).

5) Parity Rule (Francia ↔ Germania lock):
   • For every MEM–FRANCIA–WAR–{X}, create or explicitly waive a
     MEM–GERMANIA–WAR–{X} counterpart (and vice versa) with a one-line note.
   • For GEO: ensure each civ has at least two “spine” interfaces:
     one depth axis + one maritime/interface axis.

No analysis is performed here; this is a registry and parity-control surface.
