# Historical Content

This directory contains the structured historical content of the Civilization Memory Codex (CMC).

## Directory Structure

- **civilizations/**: Civilization-specific historical content
- **scholar/**: Learning accumulations and procedural cognition
- **archive/**: Archived and historical content

## Content Organization

### Civilizations Directory

Each civilization subdirectory contains structured historical content following the CMC taxonomy:

- `CIV–CORE–[CIV].md`: Civilization axioms and continuity engines
- `CIV–INDEX–[CIV].md`: Canonical registries and navigation
- `CIV–SCHOLAR–[CIV].md`: Learning accumulation ledgers
- `MEM–[CIV]–*.md`: Atomic historical memory objects
- `CIV–DOCTRINE–[CIV].md`: Frozen synthesis outputs (when applicable)
