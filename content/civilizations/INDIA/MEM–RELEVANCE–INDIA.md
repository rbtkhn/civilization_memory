Purpose: Maps STATE-relevant analytical dimensions to MEM files

This index enables the Decision Points activity and other STATE
analyses to quickly identify which MEM files contain relevant historical
parallels, patterns, and precedents for the current decision space.

Organised by the analytical dimensions that map to CIV–CORE–INDIA
diagnostics, CIV–SCHOLAR–INDIA syntheses (e.g. RLL–INDIA–0001, 0002),
and typical India decision spaces (strategic autonomy, border/interface,
colonial legacy, mechanism failure). Each dimension lists MEMs ranked
by relevance (primary = central to the dimension; secondary = relevant
context).

────────────────────────────────────────────────────────────
HOW TO USE
────────────────────────────────────────────────────────────
1. Identify the analytical dimension of the current Decision Point
   (e.g. "strategic autonomy" or "geography / heartland survivability")
2. Consult the relevant section below
3. Load primary MEMs first; secondary MEMs for depth
4. Cross-reference: a Decision Point may span multiple dimensions
5. Read MEM file contents; do not cite by name only (grounded parallels)
6. After loading primary MEMs, read their MEM CONNECTIONS and load
   up to 2 additional MEMs from connection edges when relevant
