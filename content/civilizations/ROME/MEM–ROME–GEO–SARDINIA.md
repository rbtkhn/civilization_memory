────────────────────────────────────────────────────────────
I. MEMORY PURPOSE & SCOPE
────────────────────────────────────────────────────────────
This file preserves Sardinia as **Rome's western stabilizer**—an island
combining buffer-zone function with extractive utility. Sardinia occupies
a middle position between Sicily's supply role and Corsica's resistance.

Sardinia is retained because its loss would destabilize western maritime
coherence—held for geometric value even when integration was incomplete.

────────────────────────────────────────────────────────────
II. MEARSHEIMER STRATEGIC ANALYSIS
────────────────────────────────────────────────────────────

II.A. STOPPING POWER — WESTERN MEDITERRANEAN BUFFER

Sardinia provides **western Mediterranean buffering**:
• Shields Italy from western approaches
• Supports patrols and anti-piracy
• Anchors secondary logistics
• Position matters more than integration

II.B. REGIONAL HEGEMONY — BUFFER BEFORE BENEFIT

Rome holds Sardinia for **strategic geometry**:
• Acquired after First Punic War (238 BC)
• Denial of western approaches to rivals
• Extraction (grain, timber, minerals) secondary
• Strategic value justifies governance cost

II.C. MANAGED DIFFICULTY

Sardinia demonstrates **managed difficulty**:
• Extraction coexists with unrest
• Coastal control stronger than interior
• Periodic enforcement required
• Worth retaining despite strain
