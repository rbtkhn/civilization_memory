I. PURPOSE & FUNCTION
────────────────────────────────────────────────────────────
This ledger consolidates all sources cited across MEM–FRANCIA files,
providing:

1. SOURCE WEIGHT ANALYSIS
   Which authors anchor the corpus and with what frequency.

2. ARC-T COMPLIANCE AUDIT
   Confirms sources align with CIV–ARC–FRANCIA v2.0 approved author lists.

3. DISCOVERY FLAG TRACKING
   Aggregates all ARC–DISCOVERY expansions for review.

4. GAP DETECTION
   Exposes under-represented temporal periods or source types.

5. CROSS-REFERENCE INDEX
   Maps sources to citing MEM files for traceability.
